Caddy Vault

Download & Install

Get Caddy Vault running in 2 minutes

1

Download CaddyVault

Get the complete extension package. Everything you need is included.

Download (26 KB) View on GitHub

Or clone:

git clone https://github.com/your-username/caddyvault.git
2

Run the Setup Script

Extract the ZIP, then run the one-line setup to generate icons automatically.

Mac/Linux:

bash setup.sh

Windows:

setup.bat
3

Load into Chrome

  1. Open chrome://extensions/
  2. Turn ON Developer mode (top-right)
  3. Click Load unpacked
  4. Select the caddyvault/ folder
That's it! The CV icon appears in your toolbar. Done!

Troubleshooting

Icons not showing?

Make sure the 3 PNG files are in the icons/ folder and named exactly: icon16.png, icon48.png, icon128.png

Extension won't load?

Verify that Developer mode is ON and that you selected the caddyvault/ folder (not the parent folder).

Still stuck?

Open an issue on GitHub or check the full documentation.
